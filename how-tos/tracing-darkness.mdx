### Overview

This documentation covers the letter darkness feature in DittoSheets, which allows users to customize the opacity of tracing words. Users can adjust the tracing darkness to enhance visibility and usability for tracing activities.

**Default Tracing Darkness Setting** [0:10](https://loom.com/share/21897fd16f4a43b6a9dc248cf3075005?t=10)

- The default tracing opacity is set to **20%** black.
- This opacity provides a balance between visibility and ease of writing over the text.

**Adjusting Tracing Darkness** [0:39](https://loom.com/share/21897fd16f4a43b6a9dc248cf3075005?t=39)

- Users can customize the tracing darkness using a slider located in the body section above the text.
- The slider is set to **20%** by default but can be adjusted to any value between **0%** and **100%**:
  - **Darker Example:** Setting the slider to **65%** results in significantly darker text.
  - **Lighter Example:** Setting the slider to **5%** makes the text much lighter, enhancing customization options.

**Saving Custom Settings** [1:07](https://loom.com/share/21897fd16f4a43b6a9dc248cf3075005?t=67)

- Users on the Plus Plan can save their customized tracing darkness settings.
- This allows for automatic recall of preferred settings for future use, enhancing user experience.
